# 基于Vicuna的游戏智能NPC模型

## 介绍

大语言模型的出现，使得游戏中的NPC可能具有更高的智能，突破以往硬编码的对话模式。

本模型基于[Chinese-Vicuna](https://github.com/Facico/Chinese-Vicuna)，针对游戏世界观进行微调(fine-tune)，并模拟游戏世界中的一个NPC和玩家展开对话。
